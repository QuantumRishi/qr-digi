# qr-digi

Digital asset management and content processing for the Quantum Rishi platform.

## Purpose in Cross-Repo Orchestration

The qr-digi repository handles digital asset management and content processing within the Quantum Rishi ecosystem. It provides tools for managing digital content and assets, integrating with qr-brand for brand assets, qr-data for content storage, qr-embeddings for content similarity, qr-knowledge for content knowledge bases, qr-frontend-console for content management interfaces, and qr-templates for digital templates.
